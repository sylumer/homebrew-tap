# Overview
This is a set of [Homebrew](https://brew.sh) formulas for installing things I have created.

You can "tap" this repository using the following Homebrew command:

```
brew tap sylumer/tap
```

You can list available Homebrew formulae in the tap using:

```
brew search sylumer/tap
```

# Formulae
The following formulae are currently available via this tap/repo.

- `pan` - a helper for using Piper TTS.
